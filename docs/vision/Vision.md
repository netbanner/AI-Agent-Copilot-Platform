# 1. Vision

Build an enterprise-grade AI Agent & Copilot Platform that enables developers and enterprises to build, deploy, manage and scale AI applications through a unified platform.

打造一个企业级 AI Agent 与 Copilot 平台，为开发者和企业提供统一的大模型、知识库、Agent、Workflow、工具调用以及 AI 应用开发能力。

---

# 2. Mission

本项目不是为了学习某一个框架。

而是希望通过持续开发一个真实的软件产品，逐步掌握：

- Java 企业级开发
- Spring AI
- AI Agent
- RAG
- MCP
- Workflow
- 企业架构
- 云原生
- DevOps
- Web3 Integration

最终成长为：

AI Platform Architect。
