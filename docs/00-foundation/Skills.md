# Learning Philosophy

学习不是为了掌握某个框架。

学习的唯一目的：

帮助完成 AI Agent & Copilot Platform。

每学习一个知识，

都必须回答：

1. 它解决什么问题？
2. 它属于哪个模块？
3. 我是否已经应用到项目？

如果没有应用到项目，

说明还没有真正掌握。

# Learning Rule

每学习一个技能：

必须完成：

✓ 官方文档

✓ Demo

✓ AI Platform 项目实践

✓ Markdown 总结

✓ Code Review

✓ Refactor

否则：

不能把 Current Level 提高。

# Graduation Rule

一个技能达到 L5 的标准：

实现

优化

重构

讲解

并应用到 AI Agent Platform 中。
